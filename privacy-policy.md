# Privacy Policy for CodeBuddy AI

Last Updated: June 2026

## Overview

CodeBuddy AI is a browser extension designed to assist users with competitive programming and coding interview preparation through AI-powered guidance, productivity tools, and focus features.

We value your privacy and are committed to being transparent about how data is handled.

## Information We Collect

### User Settings

The extension may store:

* User preferences
* Focus mode settings
* Session statistics
* Extension configuration
* User-provided API keys

This information is stored locally in the browser using Chrome Extension Storage.

### Website Content

When the extension is active on supported coding platforms, it may access:

* Problem statements
* Coding-related page content
* Programming language selections

This access is used solely to provide coding assistance and extension features.

Supported platforms include:

* LeetCode
* Codeforces
* CodeChef
* AtCoder
* HackerRank
* GeeksforGeeks

### AI Requests

When users request AI assistance, relevant coding-related content may be sent to AI service providers such as Google Gemini to generate responses.

## How We Use Information

Information is used only to:

* Provide AI-powered coding assistance
* Generate hints and explanations
* Enable focus mode functionality
* Track productivity statistics
* Save extension preferences
* Improve user experience

## Data Sharing

CodeBuddy AI does not sell personal information.

Data may be transmitted only when necessary to:

* AI service providers used by the extension
* Backend services required for extension functionality

## Data Storage

Most extension settings are stored locally in the user's browser.

Any backend-stored data is used solely for providing extension features and is not shared with third parties for advertising purposes.

## Security

Reasonable measures are taken to protect user information and prevent unauthorized access.

## User Control

Users may:

* Disable the extension at any time
* Remove stored extension data
* Uninstall the extension
* Remove saved API keys

## Children's Privacy

CodeBuddy AI is not intended for children under 13 years of age.

## Changes to This Policy

This Privacy Policy may be updated periodically. Updates will be reflected on this page.
